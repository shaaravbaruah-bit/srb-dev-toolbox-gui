## 一、项目概述

### 1.1 项目简介

这是一个使用 **Java Swing** 构建的多用途开发者工具箱桌面应用程序，将常用的开发者工具集成在一个界面中。

### 1.2 技术栈

- **语言**：Java
- **GUI 框架**：Java Swing
- **外观主题**：Nimbus LookAndFeel
- **依赖**：无外部依赖，全部使用 Java 标准库

- **输入**：普通 URL 或已编码的 URL

## 四、潜在 Bug 和问题分析

### 4.1 严重问题

#### Bug 1：JSON Formatter 使用字符串替换而非真正解析

**位置**：`JsonFormatter.java:34-52`

**问题代码**：

```java
formatBtn.addActionListener(e -> {
    String json = input.getText();
    try {
        json = json.replace("{", "{\n");
        json = json.replace("}", "\n}");
        json = json.replace(",", ",\n");
        output.setText(json);
    } catch (Exception ex) {
        output.setText("Invalid JSON");
    }
});
```

**问题分析**：

1. **根本不是 JSON 解析**：使用简单的字符串替换，完全不理解 JSON 结构
2. **会破坏有效数据**：如果 JSON 字符串中包含 `{`、`}`、`,`，这些字符也会被替换
3. **没有缩进**：只是添加换行，没有正确的缩进
4. **try-catch 是摆设**：字符串替换不会抛出异常，catch 块永远不会执行
5. **错误提示误导**："Invalid JSON" 提示具有误导性，因为实际上没有验证 JSON

**示例**：
输入：`{"name": "test{value}"}`
错误输出：

```
{
"name": "test{
value}
"
}
```

***

#### Bug 2：Base64 解码无异常处理

**位置**：`Base64Tool.java:36-40`

**问题代码**：

```java
decodeBtn.addActionListener(e -> {
    String text = input.getText();
    String decoded = new String(Base64.getDecoder().decode(text));
    output.setText(decoded);
});
```

**问题分析**：

- 如果输入不是有效的 Base64 字符串，`Base64.getDecoder().decode()` 会抛出 `IllegalArgumentException`
- 没有 try-catch 处理，会导致程序崩溃

**触发场景**：输入 `hello!@#$` 并点击 Decode

***

### 4.2 安全问题

#### Bug 3：密码生成器使用不安全的随机数

**位置**：`PasswordGenerator.java:42`

**问题代码**：

```java
Random rand = new Random();
```

**问题分析**：

- `java.util.Random` 是**伪随机数生成器**，不是密码学安全的
- 种子基于系统时间，可被预测
- 对于密码生成场景，应该使用 `SecureRandom`

**安全风险**：生成的密码可能被攻击者预测

***

### 4.3 跨平台一致性问题

#### Bug 4：哈希生成使用平台默认字符集

**位置**：`HashGenerator.java:36`

**问题代码**：

```java
byte[] hash = md.digest(input.getText().getBytes());
```

**问题分析**：

- `getBytes()` 不指定字符集时，使用**平台默认字符集**
- Windows 默认可能是 GBK，Linux 默认是 UTF-8
- 相同的输入在不同系统上可能产生**不同的哈希值**

**正确做法**：

```java
byte[] hash = md.digest(input.getText().getBytes(StandardCharsets.UTF_8));
```

### 5.4 核心类设计

#### 5.4.1 工具基类 (BaseTool)

```python
from abc import ABC, abstractmethod
from PyQt6.QtWidgets import QWidget, QMessageBox

class BaseTool(QWidget, ABC):
    """所有工具的基类，提供通用功能"""
    
    def __init__(self, parent=None):
        super().__init__(parent)
        self._init_ui()
        self._connect_signals()
    
    @abstractmethod
    def _init_ui(self):
        """初始化 UI，子类必须实现"""
        pass
    
    @abstractmethod
    def _connect_signals(self):
        """连接信号槽，子类必须实现"""
        pass
    
    def show_error(self, message: str):
        """显示错误对话框"""
        QMessageBox.critical(self, "错误", message)
    
    def show_info(self, message: str):
        """显示信息对话框"""
        QMessageBox.information(self, "提示", message)
    
    def clear_all(self):
        """清除所有输入输出，子类可重写"""
        pass
```

#### 原项目 JSON 格式化（有问题）

```java
json = json.replace("{", "{\n");
json = json.replace("}", "\n}");
json = json.replace(",", ",\n");
```

#### 推荐的 Python 实现

```python
import json

def format_json(input_str: str, indent: int = 2) -> str:
    """格式化 JSON 字符串"""
    try:
        parsed = json.loads(input_str)
        return json.dumps(parsed, indent=indent, ensure_ascii=False, sort_keys=False)
    except json.JSONDecodeError as e:
        raise ValueError(f"JSON 格式错误: {e}")
```
