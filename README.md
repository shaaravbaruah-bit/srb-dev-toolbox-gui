# Dev Toolbox GUI 🧰

A multi-utility **Java Swing desktop application** that provides commonly used developer tools in one place.

## 🚀 Features

- JSON Formatter
- Base64 Encoder / Decoder
- Hash Generator (MD5 / SHA-256)
- Password Generator
- URL Encoder / Decoder

## 🖥️ Built With

- Java
- Java Swing (GUI)
